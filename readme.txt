Set the fav-icon in your wordpress just by putting a link.

== Description ==

We try to make it easy and simple. So, you can make any types of image as your fav-icon in your
wordpress website. You can put any link either from website or from media library. Just put the link of image and save it.
It's Done.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `easy-set-favicon.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates

== Frequently Asked Questions ==

= Media Link is possible for fav-icon =

Yes, possible. You  can upload image to media. Then, just take link and put that as fav-icon

= Can I put a like from another website =

Yes, you can. You can put any valid link.
